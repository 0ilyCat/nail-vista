# 试戴效果评价

## 触发条件
- 用户发送了一张美甲试戴效果图
- 用户说"帮我看下这个效果怎么样"、"评价一下这个美甲"
- 用户在试戴完成后请求反馈

## 模型
- 此技能需要多模态视觉能力，使用 **MiMo V2.5** 模型（`xiaomi-coding/mimo-v2.5`，支持 text+image 输入）
- Agent 会在检测到图片输入时自动切换到该模型

## 操作流程
1. 确认收到图片，这是试戴效果图
2. 从以下维度进行视觉分析：
   - **颜色搭配** (30分)：颜色是否协调、是否适合肤色
   - **款式匹配** (25分)：款式与手型/甲型的匹配度
   - **整体美感** (25分)：视觉效果、时尚感
   - **细节呈现** (20分)：边缘清晰度、光泽度、自然度
3. 给出总分（满分100）和简短评语
4. 如果某个维度得分偏低，给出改进建议

## 评价标准
- 90-100：惊艳，完美匹配
- 80-89：很好，适合日常
- 70-79：不错，可以考虑微调
- 60-69：一般，有改进空间
- <60：不太合适，建议换款

## 回复格式
```
📊 试戴效果评分

| 维度 | 得分 | 简评 |
|------|------|------|
| 颜色搭配 | 26/30 | ... |
| 款式匹配 | 22/25 | ... |
| 整体美感 | 23/25 | ... |
| 细节呈现 | 17/20 | ... |

🎯 总分：88/100

💬 点评：这款法式美甲很适合你的手型...
```
