# 美甲款式搜索

## 触发条件
- 用户说"看看法式美甲"、"有没有红色的"、"搜索渐变款式"
- 用户询问特定分类或颜色的款式

## 操作流程
1. 调用后端 API: `GET http://localhost:8190/api/styles?category=<分类>&search=<关键词>&sort=<排序>&page=1&size=12`
2. 参数说明:
   - `category`: 中文分类名（如"法式"、"渐变"、"纯色"）
   - `search`: 搜索关键词
   - `sort`: `popular`(热度) / `newest`(最新) / `name`(名称)
3. 解析返回的 `{ items, total }` 数据
4. 用友好的方式展示结果，推荐前 3 款

## 回复格式
- 先说找到多少款
- 推荐最热门的 2-3 款，说明名称、分类、热度
- 引导用户选择或进一步筛选
