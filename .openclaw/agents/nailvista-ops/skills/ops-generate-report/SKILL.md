# 运营报告生成

## 触发条件
- 用户说"生成日报"、"写周报"、"分析趋势出报告"
- 用户需要一份结构化的运营分析文档

## 操作流程
1. 确认报告类型: `daily`(日报) / `trend`(趋势分析) / `strategy`(运营策略)
2. `curl -X POST http://localhost:8190/api/operations/reports/generate?report_type=daily`
3. 将后端返回的报告内容呈现给用户
4. 如果后端报告生成较慢，先告知"正在生成..."

## 回复格式
- 直接呈现报告内容
- 如果是日报，包含：日期、关键指标、热门款式、异常警示、建议
- 可以补充自己的数据洞察
