# 趋势数据查询

## 触发条件
- 用户说"看看最近 7 天趋势"、"试戴量走势怎么样"
- 需要分析数据变化

## 操作流程
1. `curl http://localhost:8190/api/analytics/trends?days=7`
2. 解析返回的时间序列数据
3. 分析趋势：上升/下降/平稳，找出峰值和低谷日期
4. 关注异常点（突然飙升或骤降）

## 回复格式
```
📈 7日趋势分析
• 试戴量：整体↑15%，周三最高(156次)，周日最低(89次)
• 浏览量：平稳，日均 420 次
• 收藏量：本周新增 67 次收藏

⚠️ 注意：5月20日试戴量骤降 30%，建议检查当日是否有服务异常
```
